PEAR Channel @ Michael Donat

Registering the channel: 
pear channel-discover pear.michaeldonat.net
Listing available packages: 
pear remote-list -c thornag
Installing a package: 
pear install thornag/package_name
Installing a specific version/stability: 
pear install thornag/package_name-1.0.0
pear install thornag/package_name-beta
Receiving updates via a feed: 
http://pear.michaeldonat.net/feed.xml
